
Screenshot Plugin
-----------------

Implementiert:

 1. Kopieren einer Sektion und des Fensters der CSS-Instanz.
 
 2. Versenden des Bildes an eine Person oder an das e-Logbuch als Mail mit Anhang.
 
 3. Bild, das im Fenster angezeigt wird, löschen(nur die _Darstellung_ des Bildes wird gelöscht)
 
 4. Drucken

 
Implementiert, aber noch nicht nutzbar:
 
 1. Restart
    Um den Bildschirm oder Fenster anderer Applikationen unabhängig von CSS einzulesen, kann ein zeitlich
    definierbarer Neustart des Plugins durchgeführt werden. Während der kurzen Wartezeit bis zum erneuten
    Öffnen des Fenster, können dann andere Applikationen in den Vordergrund geholt werden.
 	Erzeugt noch Probleme, wenn während der Phase des Neustarts der Fokus auf ein Menü liegt.


Geplant:

 1. Einfache Werkzeuge zur Bearbeitung des Bildes(Text hinzufügen, Löschen von Ausschnitten, Übermalen)
    
 2. Bilder, die in der Zwischenablage enthalten sind, in das Plugin übernehmen.
 
 3. Abspeichern des Bildes auf den lokalen Rechner(unterschiedliche Bildformate: JPG, BMP, TIFF, GIF)
